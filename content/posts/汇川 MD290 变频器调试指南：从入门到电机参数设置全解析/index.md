汇川 **MD290系列** 变频器是工业控制中常用的通用型变频器。本文将为您详细梳理其键盘操作、参数设置流程以及核心电机控制参数，帮助您快速上手调试。

<!--more-->

## 一、 键盘按键与功能说明

MD290 的操作面板是人机交互的核心，以下是各按键的功能定义：

| 按键 | 名称 | 功能说明 |
| :--- | :--- | :--- |
| **PRG** | 编程键 | 一级菜单进入或退出。 |
| **ENTER** | 确认键 | 逐级进入菜单画面、设定参数确认。 |
| **▲ / ▼** | 递增/递减键 | 数据或功能码的数值增减。 |
| **▶** | 移位键 | 停机/运行界面切换显示参数；修改参数时选择修改位。 |
| **RUN** | 运行键 | 在键盘操作模式下，用于启动变频器运行。 |
| **STOP/RES** | 停止/复位键 | 运行状态下停机；故障状态下用于复位操作。 |
| **MF.K** | 多功能键 | 可通过 **F7-01** 定义为正反转切换、命令源切换或点动功能。 |
| **QUICK** | 快捷键 | 根据 **FP-03** 设定切换菜单模式（如快速菜单或用户更改菜单）。 |

## 二、 参数设置流程

MD290 采用三级菜单结构进行参数设定：
1.  **一级菜单**：功能代码组（如 F0、F1 等）。
2.  **二级菜单**：功能代码标号（如 F0-03、F1-01 等）。
3.  **三级菜单**：功能代码设定值。

**操作示例**：将频率由 50.00Hz 改为 10.00Hz：
`PRG` -> `F0` -> `ENTER` -> `F0-08` -> `ENTER` -> 修改数值为 `10.00` -> `ENTER`。

## 三、 核心电机参数 (F1 组)

在驱动电机前，必须按照电机铭牌输入以下参数：

*   **F1-00**：电机类型（0：普通异步电机；1：变频异步电机）。
*   **F1-01**：额定功率 (kW)。
*   **F1-02**：额定电压 (V)。
*   **F1-03**：额定电流 (A)。
*   **F1-04**：额定频率 (Hz)。
*   **F1-05**：额定转速 (RPM)。
*   **F1-37**：**电机参数自学习**。设定为 1（静止调谐）或 2（动态调谐）后按 RUN 键启动。

## 四、 运行控制与频率设定 (F0 组)

### 1. 控制方式与启停
*   **控制方式 (F0-01)**：默认为 **2：V/F 控制**。
*   **命令源选择 (F0-02)**：
    *   **0**：操作面板命令通道（键盘控制）。
    *   **1**：端子命令通道（外部开关控制 DI）。
    *   **2**：通讯命令通道。

### 2. 频率设置 (F0-03)
MD290 支持多种主频率给定源：
*   **0 / 1**：数字设定（通过 F0-08 修改）。
*   **2 / 3**：模拟量输入（AI1 / AI2，如外接电位器）。
*   **5**：脉冲设定（DI5 端子输入）。
*   **6**：多段指令（多段速运行）。
*   **9**：通讯给定。

## 五、 停车方式与时间设置

*   **加速时间 (F0-17)**：从 0Hz 加速到基准频率的时间。
*   **减速时间 (F0-18)**：从基准频率减速到 0Hz 的时间。
*   **停机方式 (F6-10)**：
    *   **0：减速停车**（按减速时间降至 0Hz）。
    *   **1：自由停车**（立即切断输出，电机靠惯性停止）。

## 六、 正反转与多功能控制

*   **运行方向设定 (F0-09)**：0 为正向，1 为反向。通过修改此值可直接改变电机转向。
*   **端子命令方式 (F4-11)**：
    *   **0**：两线式 1（DI1 正转，DI2 反转）。
    *   **2**：三线式 1（使用按钮进行启、停、换向控制）。
*   **正反转切换**：可将 **MF.K** 键功能定义为正反转切换（**F7-01 = 2**）。

---
**提示**：在进行任何接线更改前，请确保变频器已完全断电且指示灯熄灭，以确保人身安全。

### 补充说明：
1.  **电机自学习**：强烈建议在首次调试时进行完整自辨识（F1-37=2），这能显著提升 V/F 控制的稳速精度和启动转矩。
2.  **快捷调试**：如果需要查看修改过的参数，可以利用 **QUICK 键** 进入 `--C--` 菜单模式，变频器会自动列出当前设定值与出厂值不同的所有功能码。
