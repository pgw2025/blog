# Flex 布局完全指南：从入门到精通

在 Flex 布局出现之前，网页布局主要依靠 `float`（浮动）和 `position`（定位）。但这些方法在处理“垂直居中”、“等高排列”等需求时非常痛苦。

**Flex (Flexible Box)** 意为“弹性盒子”，它是目前网页布局的首选方案。只要你掌握了它的规律，复杂的布局也能一气呵成。

---

## 一、 核心概念：容器与项目

Flex 布局就像是一个“装箱”的过程：
- **Flex 容器 (Flex Container)**：父元素，设置了 `display: flex;`。
- **Flex 项目 (Flex Item)**：容器里的直接子元素，会自动变成“弹性项目”。

### 两根轴线（至关重要）
1. **主轴 (Main Axis)**：默认是水平方向（从左到右）。
2. **侧轴 (Cross Axis)**：默认是垂直方向（从上到下）。

> **秘籍**：Flex 所有的对齐属性，都是围绕这两根轴展开的。

---

## 二、 容器属性（给父元素设置）

这些属性决定了箱子里的东西怎么摆放。

### 1. `flex-direction`：决定主轴方向
- `row`（默认）：水平从左到右。
- `row-reverse`：水平从右到左。
- `column`：垂直从上到下。
- `column-reverse`：垂直从下到上。

### 2. `flex-wrap`：是否换行
- `nowrap`（默认）：不换行，哪怕挤变形也不换。
- `wrap`：换行，第一行在上方。

### 3. `justify-content`：主轴上的对齐方式（常用）
- `flex-start`：左对齐。
- `flex-end`：右对齐。
- **`center`**：居中对齐。
- `space-between`：两端对齐，中间间隔相等。
- `space-around`：每个项目两侧间隔相等。

### 4. `align-items`：侧轴上的对齐方式
- `flex-start`：顶部对齐。
- `flex-end`：底部对齐。
- **`center`**：垂直居中对齐。
- `stretch`（默认）：如果项目未设置高度，将撑满整个容器。

---

## 三、 项目属性（给子元素设置）

这些属性决定了单个物品如何分配空间。

### 1. `flex-grow`：放大比例
- 默认 `0`（不放大）。
- 如果设为 `1`，项目将瓜分剩余的所有空间。

### 2. `flex-shrink`：缩小比例
- 默认 `1`（空间不足时自动缩小）。
- 设为 `0` 则表示死活不缩小。

### 3. `flex-basis`：初始大小
- 在分配多余空间之前，项目占据的主轴空间。

### 4. `flex`（推荐缩写）
这是项目属性中最常用的属性，它是 `flex-grow`、`flex-shrink` 和 `flex-basis` 的简写形式。

**语法结构：**
`flex: [flex-grow] [flex-shrink] [flex-basis];`

- **默认值**：`0 1 auto`（即：不放大、可缩小、基础大小由内容决定）。
- **W3C 建议**：优先使用此缩写属性，而不是单独设置三个分属性，因为缩写会自动处理一些浏览器默认值的兼容性。

**常用快捷写法：**
- **`flex: 1;`**：等同于 `flex: 1 1 0%;`。这会让项目**平分剩余空间**，常用于自适应布局（如左侧固定，右侧 `flex: 1` 占满）。
- **`flex: auto;`**：等同于 `flex: 1 1 auto;`。项目会根据内容大小展示，但也会瓜分剩余空间。
- **`flex: none;`**：等同于 `flex: 0 0 auto;`。项目**既不放大也不缩小**，保持其原始大小。
- **`flex: initial;`**：等同于默认值 `0 1 auto`。

### 5. `align-self`：个别对待
- 允许单个项目有与其它项目不一样的对齐方式，可覆盖父元素的 `align-items`。

---

## 四、 经典实战：终极垂直居中

以前写死也写不出来的垂直居中，现在只需要三行代码：

```css
.container {
    display: flex;
    justify-content: center; /* 水平居中 */
    align-items: center;     /* 垂直居中 */
    height: 100vh;           /* 容器高度占满屏幕 */
}
```

---

## 五、 总结图表

| 属性 | 作用对象 | 核心作用 |
| :--- | :--- | :--- |
| `display: flex` | 容器 | 开启 Flex 模式 |
| `flex-direction` | 容器 | 决定排队方向（横着还是竖着） |
| `justify-content` | 容器 | 主轴对齐（水平对齐） |
| `align-items` | 容器 | 侧轴对齐（垂直对齐） |
| `flex` | 项目 | 怎么分剩下的空间 |
| `order` | 项目 | 谁排在前面（数值越小越靠前） |

---

## 六、 结语

Flex 布局是前端开发者的“基本功”。只要理解了**主轴**和**侧轴**的概念，剩下的属性不过是在控制元素在这两根轴上的位置和缩放。

多写几个 `flex: 1` 和 `center`，你会发现布局从未如此简单！
